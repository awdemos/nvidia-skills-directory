# NVIDIA Skills Directory

Use this skill when the user asks about official NVIDIA skills, NVIDIA product capabilities, or which skill to use for a specific task.

## When to activate

- User asks "which NVIDIA skill does X?"
- User wants to find a skill for a product (cuOpt, NeMo, TAO, DeepStream, etc.).
- User asks about the structure, license, or entry point of an official NVIDIA skill.
- User wants to know whether a given NVIDIA tool has an associated skill.

## How to answer

1. Consult the canonical directory at `DIRECTORY.md` in this repository or at `https://github.com/awdemos/nvidia-skills-directory/blob/main/DIRECTORY.md`.
2. Use the stable anchors `## skill-<slug>` to jump directly to a skill entry.
3. Use the **Category Index**, **Product Index**, or **Alphabetical Index** to narrow down candidates.
4. Return the skill name, slug, product, primary category, license, and a one-sentence description. Include the deep link to the upstream `SKILL.md` on GitHub.
5. If no matching skill exists, say so clearly and note that the catalog is pinned to `NVIDIA/skills@366564ddf68ad55b3c12a2faee3d2fd3d3de3b36`.

## Scope limits

- This directory covers **agent skills**, not MCP servers.
- If the user asks for an NVIDIA MCP server, explain that none were found in `NVIDIA/skills` or this workspace.
